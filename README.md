# Nut Sort

Мобильная браузерная головоломка по сортировке цветных гаек.

## Как играть

1. Нажмите на стержень, чтобы выбрать верхнюю гайку.
2. Нажмите на другой стержень, чтобы перенести гайку.
3. Гайку можно положить на пустой стержень или на гайку того же цвета.
4. Соберите на каждом заполненном стержне гайки одного цвета.

## Возможности

- адаптивный интерфейс для телефона и компьютера;
- процедурно создаваемые уровни;
- отмена последнего хода;
- подсказки;
- перезапуск уровня;
- сохранение текущего уровня в браузере;
- публикация через GitHub Pages.

Игра полностью работает в одном файле `index.html` без сборки и зависимостей.
